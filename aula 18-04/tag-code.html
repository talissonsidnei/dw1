Tags

Code

Definição e uso

O que é a tag code em HTML é usada para envolver e formatar partes de texto que representam código de programação. Ela é útil para diferenciar o código do restante do conteúdo da página, tornando-o mais legível e identificável para os desenvolvedores.Quando você envolve um trecho de código com a tag code, o navegador normalmente o exibe em uma fonte monoespaçada (onde cada caractere tem a mesma largura) e, muitas vezes, também aplica uma formatação de cor diferente para distinguir o código do texto normal.

Exemplo

console.log("Olá, mundo!");

Samp

Definição e uso

A tag samp é uma das muitas tags em HTML (Hypertext Markup Language) usadas para marcar o conteúdo de uma página da web. Ela é usada para representar amostras de saída de programas de computador ou outros dados que devem ser exibidos em um formato pré-formatado. tag samp é usada para envolver o texto que deve ser exibido como uma saída de programa, como o resultado de um código-fonte, um comando de terminal ou qualquer outro tipo de saída de dados que não deve ser interpretado como código HTML. Ela é útil quando você deseja mostrar ao usuário como um determinado comando ou código irá aparecer quando for executado, sem que o navegador tente interpretá-lo como HTML.

Exemplo

Resultado da operação: 42

Pre

Definição e uso

            pre em HTML é usada para definir texto pré-formatado. "Pre" é uma abreviação de            
             "preformatted" em inglês, que significa "pré-formatado".
            Essa tag é usada quando você deseja manter a formatação exata do texto, incluindo espaços em branco, quebras de linha e qualquer outro espaçamento, exatamente como está no código HTML, sem que o navegador faça ajustes automáticos.
        
             A tag pre é útil quando você deseja exibir blocos de texto, como código-fonte, poesia ou qualquer outra forma de texto que exija uma formatação específica.

Exemplo

        int main() {
            printf("Hello, world!\n");
            return 0;
        }

Kbd

Definição e uso

kbd em HTML é usada para indicar entrada de teclado do usuário, como teclas   
            pressionadas ou combinações de teclas. "Kbd" é uma abreviação de "keyboard" em inglês, que significa "teclado". 
            
            Essa tag é útil quando você deseja destacar ou representar visualmente as teclas que o usuário precisa pressionar para executar uma determinada ação, como atalhos de teclado em aplicativos ou comandos em jogos.
            
            
            
            A tag kbd geralmente renderiza o texto dentro dela em uma fonte monoespaçada (onde cada caractere tem a mesma largura) para indicar claramente que representa uma entrada de teclado. No entanto, a aparência exata pode variar dependendo do estilo CSS aplicado à página.
    
            Em resumo, a tag kbd é útil para representar entrada de teclado do usuário em páginas da web de uma maneira clara e visualmente distintiva.
